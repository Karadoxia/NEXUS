## 🆘 If Something Goes Wrong

**Workflow won't activate?**
- Check n8n is running: `docker compose ps n8n`
- Check your browser is not blocking popups
- Try refreshing the page

**Not getting Telegram messages?**
- Check bot token in `.env`: `TELEGRAM_BOT_TOKEN`
- Check chat ID in `.env`: `TELEGRAM_CHAT_ID`
- Send test: `curl -X POST https://api.telegram.org/bot${TOKEN}/sendMessage -d chat_id=${ID} -d text="test"`

**Database tables not found?**
- Already created ✓ (run by deploy script)
- Verify: `docker compose exec -T postgres psql -U nexus -d nexus_v2 -c "\dt"`
